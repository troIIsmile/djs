# trollsmile

It's the Discord bot you love, but not the admin system you hate.

## Installing

```bash
npm i
cp example.env .env
# fill out .env
sudo apt install ffmpeg
```

## Running

```bash
npm start
```

## Updating

DM the bot "-update" while it's running.

**This will overwrite local changes.**
